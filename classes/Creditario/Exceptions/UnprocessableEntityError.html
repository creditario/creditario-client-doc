Class Creditario::Exceptions::UnprocessableEntityError < StandardError

  • lib/creditario/exceptions.rb

Excepción arrojada cuando durante la creación o actualización de algún recurso, se envían datos inválidos a creditar.io

Si se rescata esta excepción, se puede acceder a su atributo server_response para obtener la respuesta de creditar.io

Attributes
[R] server_response

Contiene los detalles de la respuesta de creditar.io
